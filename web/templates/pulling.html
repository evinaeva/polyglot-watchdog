Pulling

What does this page do?

Inspect pulled page elements, filter rows, and assign annotation decisions.

  • Collect loads pull rows.
  • Filters narrow rows by URL text and element type.
  • Screenshot thumbnails open a larger preview.
  • Decision dropdown saves row-level rule choices.

This page currently uses mock pull/rule endpoints for the visible interactions.

Decision guide: IGNORE_ENTIRE_ELEMENT excludes an item, MASK_VARIABLE keeps it but masks changing values, and ALWAYS_COLLECT always includes it.
