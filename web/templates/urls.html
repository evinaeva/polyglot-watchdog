Seed URL Manager

What does this page do?

Manage the stored seed URL list for a domain. These URLs feed discovery/capture workflows.

  • Load: fetch saved URLs for the domain.
  • Replace list: overwrite saved URLs with textarea content.
  • Add to list: merge textarea URLs into saved URLs.
  • Clear/Delete: remove all URLs or individual rows.
