What does this page do?

Use this page to query localization issues by URL fragment or keyword.

  • Apply Query calls the issues endpoint and fills the table.
  • The table stays empty until you run a query.
  • In the current scaffold, results can be mock-backed depending on filters.

Issues are shown only after applying a query.
